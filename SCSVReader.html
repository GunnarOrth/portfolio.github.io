CSV Reader
java

This CSV Reader project was made to practice my Java design practices and learn how to use Java libraries like Java Swing and Java's server/client communication libraries. The program will start by first running the server, then you can run as many client programs as you want to connect to the server. Above is the GUI created by the client class.

The program takes in a CSV file for student grades and sends that information to the server to be parsed. The server will then send the parsed information back to the client to format it into a text file. Below is an example CSV input and text output file.

The program is meant to easily be added onto for other CSV applications. Currently there is only one child class of CSVReader called StudentCSVReader, but it would be easy to add other child classes extending CSVReader, which was one of the main ideas in doing this project.
