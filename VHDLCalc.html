VHDL Calculator

The VHDL calculator was the final project in a semester class learning VHDL. This was by far the most difficult class I had taken and I am extremely proud of this project because I was able to complete it while many others didn't. The idea was to create a low level processor in VHDL. The design I created was based off the image above. Unfortunately since taking this class I have lost some of the project files including the simulated circuit in VHDL. I still however, have all the assembly files used to create the simulated circuit. The processor was created using various different muxes, flip flops, state machines, and shifters.
